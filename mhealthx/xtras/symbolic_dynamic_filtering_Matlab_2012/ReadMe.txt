This set of codes implements the symbolic dynamic filtering algorithm as a 
time series feature extraction method which is presented in the followinf paper:

Bahrampour, Soheil, Asok Ray, Soumalya Sarkar, Thyagaraju Damarla, and Nasser M. Nasrabadi.
"Performance comparison of feature extraction algorithms for target detection and classification." 
Pattern Recognition Letters 34, no. 16 (2013): 2126-2134.
